# Manuals

Operator-facing manuals for the project live here. The Phase E.2 monitor and
operator manuals are pending implementation alongside the comb-replacement
brain wrapper itself. The active design spec at
[`docs/specs/2026-04-26-cfc-comb-replacement-design.md`](../specs/2026-04-26-cfc-comb-replacement-design.md)
is the current authoritative description of the architecture.

The retired v2 monitor user manual, written against the four-module
1,100-neuron architecture which preceded the comb-replacement design, has been
moved to
[`archive/legacy-v2/docs/manuals/monitor-user-manual.md`](../../archive/legacy-v2/docs/manuals/monitor-user-manual.md)
and should not be cited as authoritative.
